# 小贴纸

世界杯开赛前的某一个周末，我在新天地逛街，非常幸运地遇到了一个非常好玩的世界杯玩具：**Panini 世界杯贴纸**。

{% asset_img book.jpg Panini小册子 %}

我之前知道有球星卡，世界杯年肯定会出国家队的卡片。当我第一次在卡牌商店见到贴纸册的时候，瞬间眼前就亮了。这种一点点抽贴纸把小册子填满的快乐，感觉比以前玩二游抽卡攒图鉴要带劲很多。

这个贴纸大致是这样的：

1. 有一些 FIFA 相关的图，比如本届世界杯的 LOGO，比如历届冠军球队的合影。
2. 除了 FIFA 的图，剩下的就是每一个国家队的20张贴纸，这20张贴纸包含1张国家队 LOGO，1张国家队合影以及18名球员的大头。不过比较可惜的是这个贴图是早于国家队名单上市的，估计 Panini 是根据国家队的历史表现选择了18名有代表性的球员。所以非常可惜，虽然帕尔默和埃斯特旺在贴纸上，但是他们最终没能前往美加墨。

{% asset_img france.jpg 法国队页面 %}

作为贴纸来说，价格的确会比球星卡便宜很多。这本小册子是15元一本，一盒贴纸是300元，包含50包贴纸，每包7张。我买的时候在搞活动，买一盒贴纸会送一本册子。

{% asset_img card-pack.jpg 一大盒贴纸有50包小贴纸 %}

我买的时候才刚上市，还不是很火。等世界杯快开始的时候这个贴纸就卖断货了。

拆贴纸的时候也是会记录一些有代表性的球员，比如当代的“双骄”。

{% asset_img messi.jpg 梅老板 %}
{% asset_img ronaldo.jpg 罗哥 %}

作为一个“抽卡”性质的产品，很显然贴纸一定是会有重复和缺失的。想要靠几盒贴纸凑齐书不太现实的。好在小红书和闲鱼上都有一些互换帖子，可以做一些“场外互换”。为此我也做了一个[互换页面](/life/20260629-world-cup-2026/20260629-world-cup-2026/panini_no_exchange.html)，可以快速查询我所缺少的卡片以及我多余的卡片。感兴趣的可以查看。

# 球衣消费

作为一个足球迷，必不可少的就是买一些球衣。况且今年还是世界杯年，各国都推出了自己的2026赛季国家队球衣，我怎么能忍住不下手呢？

## 汇总

本次我一共买了三件世界杯球衣，分别是德国队主场球迷版（沃尔特玛德印号）、意大利客场球迷版（光版）和阿根廷客场球员版（光板）。

眼尖的小伙伴们可能发现了，这三件球衣被阿迪达斯垄断了，事实上我也确实是有过相关考虑的。不是我青睐阿迪达斯，而是耐克在做相关足球周边的时候真的是太不走心了。

- 首先我是一个切尔西球迷，我觉得切尔西2025/26赛季的球衣其实还挺好看的，本身不是很排斥耐克的用料和科技。但是我在看了耐克做的中超2025/26赛季全系列球衣加上切尔西2026/27赛季主场的德邦工作服之后，我真的是对耐克表示无语。
- 其次是我作为一个中国人，当我发现耐克做的2026年国足球衣还是依托的时候，我真的不知道该给耐克说什么。一方面是上一个世界杯年国足低对比度的鲜红球衣硬是搭配黑色印号，导致我在球场上完全无法靠号码分辨谁是谁。另一方面是难得市售球员版，不仅只开放了主场，还立刻售罄。就是连自家人的钱都不想赚吗？
- 没有对比就没有伤害，阿迪达斯反手就去和苏超合作，推出了极具特色的十三太保球衣。我们先不说这十三太保的衣服好不好看，单说这个设计、多样性和走心程度，已经甩了耐克十条街了。这就是联赛垄断球衣冠名的后果，美职联选中了阿迪，美职联的球衣的确是百花齐放，然后中超选择了耐克，一言难尽。

包括下一个赛季，除了国家队球衣和切尔西球衣，我是不会给耐克掏一分钱的。除非哪一天你的审美能够上去。

## 阿迪王

我来分享一下本次买的三件世界杯球衣，阿迪王世界第一。

1. 首先是我们国骂哥的球衣。这件是最早买的，2026年1月，因为准备在公司年会上将脱口秀，所以准备了一个道具。我记得是前几年沃尔特玛德在斯图加特的时候就认识了这名高中锋。因为之前德国的高中锋是菲尔克鲁格，豁牙近几年也没啥声音了，加上我也挺喜欢德国队的，所以不自觉地关注上了国骂哥的表现。而且在纽卡的时候沃尔特玛德的确打进过一些比较精彩的进球（虽然尽享最深的是东北德比首回合不敌桑德兰的那次乌龙球）。整体上德国队的三色条纹让这件球衣的色彩搭配难度降了一个等级。
{% asset_img germany.jpg 国骂哥专供 %}

2. 在一场酣畅淋漓的点球大战之后，意大利再次无缘世界杯正赛。这一场失利除了把意大利打回了家，还直接导致了意大利球衣的全线降价（是的，德国被巴拉圭淘汰后的同款待遇），1099 的球员版很贵，699 的球迷版会需要心疼一下钱包。但是两百多的球迷版我说可以无脑拿下！
{% asset_img italy.jpg 打折的就是香 %}

3. 最后选择购买阿根廷的客场球员版，一个是因为这件衣服的花纹和材质真的很不错，我觉得阿根廷的主场蓝白条纹印象非常统一，甚至会有些审美疲劳，反而是每年的客场各有特色（不像国足的客场，每年在蛋白和蛋清中间二选一）。另一个想法也是这件衣服上会配有一个2022世界杯的冠军盾，说不定今年比完之后这个冠军盾就易主了。（真的易主了，让我们恭喜二星斗牛士军团！）所以提前将其拿下。
{% asset_img argentina.jpg 最有设计感的一款，配上冠军盾 %}
