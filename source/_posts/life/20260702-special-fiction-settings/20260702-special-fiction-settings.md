# 灵感来源

今年年会的时候，我们去了滴水湖的洲际酒店。

这个酒店的布局很有意思，它的五个客房分区是一朵花的五个花瓣，可以参考下面的图片内容。

{% asset_img map.png 五栋楼的花瓣基本保持中心对称 %}

当时在里面闲逛的时候产生了一个**神奇的灵感**，有关推理故事的。会有这个想法是因为这个酒店天然就有一种会发生凶杀案的感觉：

- A~E 五个区域都可以从中心大厅前往，而且不看路牌的话真的很容易迷路；
- 这个酒店位于人工湖上，出湖只有一座桥。虽然在滴水湖中被困住很难，但是这座桥让这个酒店可以出现非常经典的“**暴风雪山庄**”设定。


# 添油加醋

我们假设这个酒店是一起凶杀案的现场，那么只要给这个豪华酒店一些神奇的设定，就能够满足一些诡计诞生的条件。以下是我根据我的阅读和脑洞随便想的。

{% asset_img indoor-scene.jpg 单个花瓣的内景 %}

1. **核心诡计**：想办法让人误会自己所在的别院，创造不在场证明。
2. 这个酒店一定是一个顶级设计师设计的，带有一些传统且现代的机关，让一些部分能在机械作用下动起来。
    1. 为了让这个设定听上去更加真实，我们只移动主楼的电梯；因为如果我们的设定是几个别馆能够移动而里面的住客没有任何感知，就显得很假。
    2. 为了把机关做在电梯里，我们可以让主楼的公共区域和客房之间保持一定的物理距离，比如：我们设定电梯在主楼的正中央，需要向上或者向下才能够到达客房区域。
        - 客房区域当然也有楼梯（安全出口）能够直达主楼区域，但是为了能够让人混淆所在区域，这些楼梯要设计地曲折一些，让人走下来的时候不会意识到自己的出口有问题。
3. 如何让住在里面的人不发现自己的别馆不是自己应该去的呢？（比如我以为我去的是 A 馆，但是实际上我在 C 区）
    1. 故事必须发生在晚上，因为窗外的风景不同一定会暴露这一点。
    2. 为了让暴风雪山庄的故事更加经典，就设计在一个暴雪的冬夜，这座山庄，建在某一处深山里，而且是在一个山顶上，四周都是地势更低的山脉，而且离最近的城市 / 居民区都很远。这么做的目的是即使有人在晚上仔细看向窗外，也很难分辨出这是错误的房间。
        - 基于这一点，也可以做一个很隐秘的破绽角度：可以设定天气预报说明当地今晚的风向，然后让不同别馆窗外的积雪厚度不一致。
    3. 几个别馆内部的装饰完全一致，只有房号，没有区域编号。那怎么证明自己去的地方是对的呢？让电梯告诉自己答案。
        - 所以凶手/帮凶一定会想办法破解电梯，给重要角色提供关键的错误信息。

{% asset_img lounge.jpg 单个花瓣的上下若干楼是通的，图片拍的是 B 区，只有三楼和四楼是通的，而 C 区则是一楼到四楼均有连接 %}

4. 这种情况下，想要跨别馆作案，一定得有凶手的小道。
    1. 比较低级的做法是：别馆之间有一些凶手才知道的密道；
    2. 比较高级的做法是：
        1. 命案发生的时间里，大厅里一直有人活动（一直关注电梯的运作情况），让大家误认为没有人乘坐电梯到达过命案现场。
        2. 实际情况确是：凶手用某一种途径，让大家错误地认为他去了自己房间/做一件和命案无关的事情，但是实际上却在大家的目送之下去了凶案现场。
        - 基于这个情况，我们需要同时地，让：
            - 死者所在的别馆只有他一个住客。（死者是个特殊人物，所以单独住也很合理）
            - 凶手会和另外某位同行者在一个别馆内，从而互相提供不在场证明。
        - 最后凶手露出的某一个**破绽**，可以是他的某处表述，和与他同馆的另一位角色的不同。

5. 小说的定位
    - 推理小说有不同的风格。我觉得这种带一定科技的建筑（有点类似于《馆系列》）肯定能让读者感受到这里面的某些手法，但是可以：
        - 让读者猜不到机关；
        - 让读者能够在阅读的时候猜到机关，但是推理不出凶手是谁。
        
        我觉得对于我的阅历来说就实现**第二种**好了。
        
    - **特点**：新奇的建筑，不在场证明，通过凶手的破绽而不是偶然性证据来破案。
